# FreeCrawl SEO

Open-source desktop SEO crawler — a free alternative to Screaming Frog.

## About

FreeCrawl SEO is a cross-platform desktop application for crawling websites and analyzing SEO metrics. It aims to provide the core functionality of commercial SEO crawlers in a free and open-source package.

## Status

Early development. Features and architecture are being actively designed.
